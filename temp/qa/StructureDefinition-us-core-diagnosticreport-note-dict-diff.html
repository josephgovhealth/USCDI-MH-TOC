    8. DiagnosticReport.code
    Definition

    The test, panel, report, or note that was ordered.

    ShortUS Core Report Code
    Comments

    The typical patterns for codes are: 1) a LOINC code either as a translation from a "local" code or as a primary code, or 2) a local code only if no suitable LOINC exists, or 3) both the local and the LOINC translation. Systems SHALL be capable of sending the local code if one exists.
