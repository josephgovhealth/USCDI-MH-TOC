This value set includes codes based on the following rules:

  • Include all codes defined in http://www.ama-assn.org/go/cpt

  • Include all codes defined in https://www.cms.gov/Medicare/Coding/HCPCSReleaseCodeSets
  • Include all codes defined in http://www.cms.gov/Medicare/Coding/ICD10
  • Include all codes defined in http://www.ada.org/cdt
  • Include all codes defined in http://loinc.org
