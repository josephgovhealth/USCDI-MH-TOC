Name:USCDIRequirement
Title:US Core USCDI Requirement Extension
Status:Active as of 2023-04-03
Definition:

This extension is only used in the US Core Implementation Guide''s Profile StructureDefinition elements. It flags elements that are Additional USCDI Requirements for certified systems. In other words, elements that are not marked as Must Support but are required for ASTP Health IT certification testing. See the Conformance page for more information.
